## Welcome to Perfeggt Timer,

### 🍳 How to Use

- Boil Water – Get that pot bubbling!
  
- Add Eggs – Gently place your eggs into the hot water.
  
- Start the Timer hit start.
  
- Wait Patiently – Let the timer do its magic!
